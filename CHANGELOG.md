# Changelog

All notable changes to RNK Alpha MapGen are documented in this file.

## [1.0.3] - 2026-04-18

### Added
- Foundry VTT v14 verified compatibility alongside v13

### Fixed
- Trap placement logic corrected for edge-case room configurations
- Module metadata cleaned and finalized for initial public release

## [1.0.0] - 2026-04-01

### Added
- All features from the Free tier
- Architecture controls: room count (3-200), level count (1-20)
- Sliders for room variety, corridor complexity, branching factor, dead-end frequency, treasure density, and secret room frequency (each 0-10)
- Room size options: Tiny, Small, Medium, Large, Huge
- Symmetry options: Asymmetric, Balanced, Symmetric
- Verticality options: Flat, Moderate, Multilevel
- Connection density options: Isolated, Moderate, Interconnected
- Chamber shape options: Square, Circular, Irregular, Mixed
- Encounter pacing options: Spread, Clustered
- Trap and hazard system with count (0-100), DC (5-30), and radius (1-30) controls
- 15 trap types: None, Random, Spike Floor, Flame Jet, Acid Spray, Scythe Blade, Pit Trap, Dart Shooter, Web Snare, Teleport (Random), Teleport (Linked), Gravity Well, Silence Aura, Alarm Bell, Magic Drain
- Deep Foundry VTT scene and canvas integration
